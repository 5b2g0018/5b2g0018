Professional 專業領域

🖥️ 資訊工程的初探

在南台資工的學習過程中，我發現程式碼不只是冰冷的指令，而是解決問題的鑰匙。目前我正專注於打好扎實的基礎，並對軟體開發與數位邏輯充滿好奇心。

🛠️ 核心技術棧 (Tech Stack)

  • 程式語言基礎： 熟練掌握 C++ 邏輯開發，並進修 Python 進行自動化腳本撰寫。
  • 網頁開發： 學習 HTML5, CSS3 響應式設計，能獨立修改並優化網頁模板。
  • 開發工具： 習慣使用 VS Code 進行開發，並練習使用 Git 進行版本管理。

🎯 專注方向

我對「前端介面設計」與「演算法邏輯」非常有興趣。我喜歡觀察一個功能是如何從無到有，透過邏輯推導最終呈現在螢幕上的過程。

「寫程式最開心的時刻，就是解決掉困擾已久的 Bug。」

🛡️ 專業態度與未來展望

身為資工的一員，我深知技術更新的速度極快，因此我保持著「持續學習」的習慣。未來我希望能深入研究不同的技術領域，並做出能真正幫助別人的應用程式。
