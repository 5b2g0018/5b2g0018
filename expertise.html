Expertise 技術專長

雖然我還沒變成大神，但我很享受那種「從 0 到 1」把功能做出來的成就感。這裡紀錄了我目前會的技能，以及我還在努力對抗的各種程式語法！

💻 程式語言 Programming

  • C / C++： 基礎邏輯與演算法實作 (核心專長)。
  • HTML5 / CSS3： 網頁佈局與響應式設計。
  • JavaScript： 基礎動態效果與互動邏輯 (學習中)。
  • Python： 基礎語法與簡單的自動化腳本。

🛠️ 開發工具與軟體

  • VS Code： 主要使用的編譯環境。
  • Git / GitHub： 基礎版本控制與專案託管。
  • Arduino： 基礎硬體控制與物聯網實驗。
  • Markdown： 撰寫技術筆記與文件。

🚀 最近正在努力的挑戰 (Currently Learning)

🎨 UI/UX 設計： 我以前以為寫程式只要功能會動就好，後來才發現，要把介面做得「好看又好用」真的超難！目前我正在自學如何排版，希望未來我寫的網頁不會讓使用者迷路。

🧠 資料結構與演算法： 雖然現在看到一堆複雜的邏輯頭會有點痛，但老師說這是資工系的靈魂。我正在練習從「暴力解法」慢慢進化到用更聰明的邏輯去解決問題，雖然偶爾會被 Bug 卡到懷疑人生，但解出來那一刻真的很爽！
